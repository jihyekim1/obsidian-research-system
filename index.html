Installation

설치 방법

순서대로 따라하면 약 10분 안에 완료됩니다.

1

Obsidian 설치

노트 관리 앱 Obsidian을 설치합니다. obsidian.md 에서 다운로드 후 설치하세요.

2

VS Code 설치

code.visualstudio.com 에서 VS Code를 다운로드 후 설치합니다.

3

VS Code 확장 설치

VS Code를 열고 Ctrl+Shift+X → 아래 두 확장을 검색해서 설치합니다.

# 확장 1 — AI 어시스턴트 (게시자: Anthropic) Claude Code # 확장 2 — Python 스크립트 실행용 (게시자: Microsoft) Python
⚠ 주의: Claude Code 사용에는 Claude Pro 이상의 구독이 필요합니다. claude.ai 에서 먼저 가입하세요.
4

저장소 다운로드

이 페이지 상단의 ⬇ 파일 다운로드 (ZIP) 버튼을 클릭해서 ZIP 파일을 받고, 원하는 위치에 압축 해제합니다.

💡 예시: C:/Users/홍길동/Documents/obsidian-research-system/ 에 압축 해제
5

VS Code에서 열고 Python 패키지 설치

VS Code에서 File → Open Folder → 방금 압축 해제한 obsidian-research-system 폴더 선택

그 다음 Terminal → New Terminal 을 열고 아래 명령어 실행:

pip install -r requirements.txt
6

setup.py 실행 — 자동 설정

같은 터미널에서 아래 명령어를 실행합니다. vault 경로와 연구 관심사를 물어봅니다.

python setup.py

입력 예시:

# vault를 만들 경로 입력 (없으면 자동 생성) Vault 경로: C:/Users/홍길동/Documents/Obsidian Vault # 논문 AI 분석 시 활용되는 연구 관심사 연구 관심사: machine learning, HCI, education technology

실행하면 vault 폴더 구조, config.txt, .api-keys, CLAUDE.md가 자동 생성됩니다.

7

API 키 입력

setup.py가 생성한 99-meta/.api-keys 파일을 열어 발급받은 키를 입력합니다.

GEMINI_API_KEY=AIza... NAVER_CLIENT_ID=abc123 NAVER_CLIENT_SECRET=xyz789
8

모니터링 키워드 설정

99-meta/paper-watch.md 파일을 열어 관심 키워드를 입력합니다.

# 한 줄에 하나씩, - 로 시작 - machine learning - educational technology - human-computer interaction
9

Obsidian에서 vault 열기

Obsidian 앱 실행 → Open folder as vault → setup.py에서 지정한 vault 경로 선택

이후 노트 확인은 Obsidian에서, Claude와 대화는 VS Code에서 합니다.

10

VS Code에서 vault 열고 Claude Code 시작

VS Code에서 File → Open Folder → vault 폴더 선택

C:/Users/홍길동/Documents/Obsidian Vault

폴더가 열리면 VS Code 하단 상태바의 Claude 아이콘을 클릭하거나, Ctrl+Shift+P → Claude Code: Open 으로 채팅창을 엽니다.

첫 메시지를 보내면 논문 수집 파이프라인이 자동으로 시작됩니다.

Troubleshooting

자주 묻는 질문

스크립트가 실행되지 않아요

Python이 설치되어 있는지, pip install -r requirements.txt를 실행했는지 확인하세요. 오류 메시지와 함께 config.txt가 없다고 나오면 setup.py를 먼저 실행하세요.

논문이 수집되지 않아요

.api-keys 파일에 GEMINI_API_KEY가 입력되어 있는지 확인하세요. Semantic Scholar는 API 키 없이도 동작하지만 rate limit이 있을 수 있습니다.

PDF 분석이 안 돼요

inbox-pdfs/에 넣은 PDF 파일명이 논문 제목과 비슷한지 확인하세요 (예: machine-learning-basics-2023.pdf). 파일명으로 기존 노트와 매칭합니다.

Claude Code가 뭔가요?

Anthropic이 만든 AI 코딩 어시스턴트입니다. VS Code 확장으로 설치하면 에디터 안에서 바로 대화할 수 있습니다. 폴더를 열면 루트의 CLAUDE.md를 자동으로 읽어 작동 방식을 설정합니다. Claude Pro 이상의 유료 구독이 필요합니다.

VS Code 확장이 안 보여요

VS Code 마켓플레이스에서 게시자가 "Anthropic"인 Claude Code를 찾아 설치하세요. 설치 후 VS Code를 재시작하고, claude.ai 계정으로 로그인하면 활성화됩니다.
