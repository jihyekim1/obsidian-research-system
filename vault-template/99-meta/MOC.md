# Meta

시스템 운영에 필요한 설정 파일들이 있는 폴더입니다.

## 주요 파일

- `config.txt` — vault 경로 및 연구 관심사 설정 (setup.py로 자동 생성)
- `.api-keys` — API 키 (절대 GitHub에 올리지 마세요!)
- `paper-watch.md` — 논문 자동 수집 키워드 목록
- `paper-seen.txt` — 이미 수집한 논문 ID 목록 (중복 방지)
- `scripts/` — 자동화 스크립트
- `inbox-pdfs/` — 수동으로 추가할 PDF를 여기에 넣으세요
